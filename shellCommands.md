# List of misc useful commands

- `sudo pcmanfm`
  - Opens File manager as admin
- `tar -xf <file name>`
  - extract contents of tar.xz file into similarly named file
- `sudo mv <file name> <location>`
  - Move file
- `sudo bash <.sh file name with extension>`
  - Run script

# Updating

- `sudo apt update`
  - Check for updates
- `sudo apt upgrade`
  - `sudo apt upgrade-all`
  - Apply updates
